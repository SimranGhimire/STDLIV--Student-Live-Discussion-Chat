# Overview
### STDLIV Student Live Discussion Chat is a real-time communication platform designed to encourage academic collaboration and intellectual exchange among students. The system provides an interactive space where learners can engage in meaningful conversations, share insights, and participate in topic-based discussion rooms.


# Features
### 1. Real-time chat communication
### 2. Topic-based discussion rooms
### 3. User authentication (PHP + SQL)
### 4. Responsive front-end design
### 5. Node.js backend with WebSocket (Socket.io) integration
### 6. Secure and organized messaging interface


# Technologies Used

## Backend:
### 1. Node.js
### 2. Socket.io
### 3. PHP
### 4. SQL / MySQL

## Frontend:
### 1. HTML
### 2. CSS
### 3. JavaScript


# Installation & Setup
## 1. Clone the Repository
### git clone https://github.com/SimranGhimire/STDLIV--Student-Live-Discussion-Chat.git

## 2. Install Node.js Dependencies
### npm install

## 3. Configure the Database
### Import the SQL file into MySQL
### Update database credentials in php/db.php

## 4. Start the Server
### node server.js

## 5. Access the Application
### http://localhost:3000


# How It Works
### 1. Users log in/register through PHP + SQL backend
### 2. The front-end connects to the Node.js server via Socket.io
### 3. Messages are broadcasted in real time to all connected users or specific rooms
### 4. The interface automatically updates with no page reload
